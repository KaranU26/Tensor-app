# GYM EQUIPMENT ROUTINES

## 1. Full Body Strength (Barbell Focus)

**Duration:** 60 minutes | **Difficulty:** Intermediate | **Equipment:** Barbell, Rack

### Workout:
```
Warm-up (5 min):
- Arm circles: 10 each direction
- Cat-cow stretches: 10 reps
- Bodyweight squats: 10 reps
- Inchworms: 8 reps

Main Workout:
1. Barbell Back Squat
   - 2 sets x 3 reps @ 85%
   - 1 set x 5+ reps @ 75%
   - Rest: 3 minutes

2. Barbell Bench Press
   - 2 sets x 3 reps @ 85%
   - 1 set x 5+ reps @ 75%
   - Rest: 3 minutes

3. Barbell Deadlift
   - 1 set x 3 reps @ 85%
   - 1 set x 5+ reps @ 75%
   - Rest: 3 minutes

4. Barbell Rows
   - 3 sets x 5 reps @ 80%
   - Rest: 2 minutes

5. Accessory Superset (3 rounds):
   - Leg Press: 8 reps
   - Cable Chest Fly: 8 reps
   - Rest: 90 seconds
```

### Post-Workout Stretching (10 min):
```
1. Hip Flexor Lunge Stretch: 30 sec each side
2. Quad Stretch (standing): 30 sec each leg
3. Hamstring Stretch (supine): 30 sec each leg
4. Chest Doorway Stretch: 45 sec
5. Shoulder Cross-Body Stretch: 30 sec each
6. Lat Stretch (overhead reach): 45 sec
7. Child's Pose (back release): 60 sec
8. Spinal Twist (supine): 30 sec each side
9. Downward Dog: 60 sec
10. Standing Forward Fold: 60 sec
```
